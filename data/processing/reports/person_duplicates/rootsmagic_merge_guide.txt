ROOTSMAGIC 10 DUPLICATE MERGING GUIDE
==================================================

STEP-BY-STEP SAFE MERGING PROCESS:
1. Import final_with_media.ged into RootsMagic 10
2. File > Backup Database (create safety backup)
3. Tools > Merge > Find Duplicate People
4. Review each suggested merge carefully
5. Use merge preview to verify data combination
6. Merge only when you're 100% confident
