## 1. RootsMagic 11 (Desktop - Windows/Mac)

### Tree Management
- Add/edit/delete persons with full event support
- Life Summary panel (snapshot of events, spouses, children, parents, siblings)
- Drag-and-drop reordering of facts and family members
- Multiple name support (alternate names, married names, nicknames)
- Proof statuses: Proven, Disproven, Disputed, **Proposed** (new - puzzle-piece icon for hypotheses)
- Associations (link non-family relationships: neighbor, witness, godparent)
- Color coding of individuals
- Bookmarks for quick access to key people
- Groups (custom lists of selected people)
- Tasks/To-Do list per person or global
- Problem alerts (data quality warnings inline)

### Charts & Visualization
- Pedigree view (multi-generation)
- Family view (nuclear family focus)
- Descendant view
- **Wall charts** (rebuilt from ground-up): ancestor, descendant, fan charts
- Box charts (book-printable, standard page sizes)
- Fan charts (new in v11)
- Timeline charts
- Relationship chart (path between two people)

### Search & Research
- **Advanced Search** with AND/OR/BUT NOT operators
- Separate rule types: events, names, attributes, family attributes, associations
- Customizable search result columns (independent from People List)
- Export search results to Excel/CSV
- WebSearch integration (Ancestry, FamilySearch, Google auto-filled)
- FamilySearch integration (read AND write)
- Ancestry TreeShare (manual sync, batch updates)
- Record hints from Ancestry and FamilySearch

### Reports & Publishing
- **Narrative Reports**: NGSQ (Modified Register), NEHGS (Register), ancestor/descendant
- **Ahnentafel** (ancestor list, narrative format)
- **Family Group Sheet**
- **Individual Summary**
- **List Reports**: Address, Ahnentafel, Birthday/Anniversary, CountyCheck, Descendant, Duplicate, Fact, Individual, Kinship, LDS, Marriage, Missing Info, Multimedia, On This Day, Place, Statistics, Surname Statistics, Timeline, WebTags, Who Was There
- Book publishing (compile multiple reports into a book)
- Export to DOCX, XLSX, SVG, PDF
- Custom report templates

### DNA Tools
- DNA matching view
- Chromosome mapping display
- DNA charts in reports

### Media Management
- Photo/document attachment to persons and events
- Multimedia list report
- Media gallery browsing

### Sources & Citations
- SourceWizard (template-based citation builder)
- Source templates (hundreds of templates based on Evidence Explained)
- Master source / citation detail split
- Free-form and template-based citation entry
- Source list reports

### Data Quality
- Problem alerts for data inconsistencies
- Duplicate detection (Duplicate List report)
- Missing Info report
- CountyCheck report (verify county matches for places/dates)
- Data clean tools (search and replace)

### Collaboration
- File sharing via GEDCOM
- Ancestry TreeShare (two-way sync)
- FamilySearch sync (read and write)
- Dropbox/cloud file sync (via file storage)

### Import/Export
- GEDCOM 5.5.1 import/export
- GEDCOM 7 support
- Ancestry TreeShare
- FamilySearch direct integration
- RootsMagic file format (.rmtree)
- Excel/CSV export of lists
- Import from other programs (PAF, Family Origins, etc.)

### AI Features
- **AI Prompt Builder** (new in v11) - generates structured prompts for ChatGPT/Claude with genealogical context

### Maps & Geography
- Place list management
- County verification
- WebSearch integration for place research

### Settings & Customization
- Customizable views, colors, fonts
- LDS/temple ordinance support (toggle on/off)
- Date format preferences
- Multiple file management
- Customizable columns everywhere

# FEATURE GAP ANALYSIS: What GedFix is MISSING

## Current GedFix Capabilities
- Person CRUD, Family CRUD, Event CRUD
- Pedigree chart (3-5 gen)
- Tree consistency checker (15+ issue types)
- Validation badges with research URLs
- Media gallery with viewer
- Global search
- Timeline view
- 5 report types
- GEDCOM import/export with living filter
- Settings screen
- Overview dashboard with stats/charts
