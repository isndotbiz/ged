# GedFix Swift (Archived)

Swift/SwiftUI + GRDB macOS-only implementation. Archived April 2026.

**Why archived:** The Tauri app (`apps/desktop/`) covers all platforms from one codebase and is significantly more complete. This was a parallel experiment — ~40% feature parity, macOS only.

**Reference value:** The Swift GEDCOM parser and GRDB schema may be useful if a native macOS App Store submission is ever needed (Apple sometimes favors native apps for store featuring). The data model field names are nearly identical to `@gedfix/core`.

**Do not delete** — contains working Swift GEDCOM parser, SwiftUI views, and GRDB migration schema.
